# Claude Code로 백지에서 만드는 법

빈 폴더에서 시작해 Claude Code로 결과물을 **단계별로 쌓아 만드는 과정**을 가르치는 HTML 강의덱입니다.

## 이 덱의 핵심 주장

결과물은 프롬프트 한 줄로 한 번에 나오지 않습니다.
빈 폴더에서 **설계 → 데이터 → 콘텐츠 → 조립 → 검증**을 단계별로 밟아 만듭니다.
잘 만들면 그 과정이 스킬·하네스로 남아 다음엔 빨라집니다.

## 백지에서 만드는 공통 순서

1. 빈 폴더를 만들고 그 안에서 `claude` 실행
2. 바로 만들지 말고, 목표를 단계로 쪼개 달라고 시킨다
3. 단계별로 파일을 만들게 하고 중간 산출물을 저장한다
4. 마지막에 조립하고, 앞 단계 정의와 맞는지 검증한다
5. 반복될 것 같으면 방법을 스킬/CLAUDE.md로 정리한다

## 두 가지 제작 패턴

- **멀티에이전트 파이프라인** (auction_mbti): 복잡한 산출물을 단계별로 만들고, 그 과정을 .claude 하네스로 승격
- **시행착오 → 스킬 박제** (auction-crwal0629): 까다로운 자동화를 뚫은 뒤 성공/실패 패턴을 스킬로 저장

## 덱에 포함된 결과물

## 파일 구조

```
index.html            # 강의덱
data/projects.json    # 각 결과물의 실제 구조 + 백지 제작 순서 데이터
```

## 새 결과물 추가법

`data/projects.json`의 `projects` 배열에 객체를 추가합니다.

- `realStructure`: 다 만들면 결국 이렇게 되는 실제 구조
- `pipeline`: 빈 폴더에서의 단계별 흐름
- `reproduce`: 빈 폴더에서 시작하는 실제 명령/프롬프트
- `keyLesson`: 이 케이스의 핵심 교훈

## 원칙

- git clone으로 가져오는 게 아니라 **백지에서 만드는 순서**를 보여준다.
- 한 번에 만들지 말고 단계로 쪼갠다.
- 디자인(폰트·색) 지시는 넣지 않는다(로컬 폰트 의존이라 재현 불안정).
