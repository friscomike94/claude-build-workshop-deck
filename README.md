# 우리 결과물처럼 만드는 정확한 레시피

빈 폴더에서 시작해 Claude Code로 4개 결과물을 만드는 초보자용 설명서입니다.

## 특징

- 단계당 한 슬라이드
- 터미널 명령, Claude Code 프롬프트, GitHub 클릭 순서 분리
- 결과물별 완성 스펙, 파일명, 데이터 구조, QA 기준 포함
- 단순 "만들어줘"가 아니라 중간 산출물과 검수 기준을 지정

## 포함된 설명서

1. auction_mbti 정확히 만드는 법 — 14단계
2. auction_crawl 정확히 만드는 법 — 15단계
3. malso_deck 정확히 만드는 법 — 10단계
4. apt_report 정확히 만드는 법 — 12단계
