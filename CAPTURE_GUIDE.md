# 화면 캡처 체크리스트

## 공통
- 16:9 브라우저 화면 기준
- 너무 많은 UI를 한 화면에 넣지 않기
- 초보자가 눌러야 하는 버튼이 보이게 캡처
- 캡처 후 파일명은 영어 소문자 + 하이픈

## Claude 캡처
1. 새 대화 시작 화면
2. 프롬프트 입력창
3. Claude가 파일 구조를 제안하는 장면
4. Claude가 코드를 생성하는 장면

## GitHub 캡처
1. repo 첫 화면
2. Add file 버튼
3. Upload files 화면
4. Commit changes 버튼
5. Settings → Pages 화면
6. 배포 URL 확인 화면
